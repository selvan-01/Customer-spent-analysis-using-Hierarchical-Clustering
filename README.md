# 🧠 Customer Spend Analysis using Hierarchical Clustering

An advanced Machine Learning project that segments customers based on their income and spending behavior using **Hierarchical Clustering (Agglomerative Clustering)**. This project is deployed as an interactive **Streamlit Web App** for real-time data analysis and visualization.

---

## 🚀 Project Overview

Customer segmentation is a crucial task for businesses to understand their audience.
This project uses **Hierarchical Clustering** to group customers into different segments based on:

* 💰 Annual Income
* 🛍 Spending Score

It helps businesses:

* Target the right customers
* Improve marketing strategies
* Increase sales and personalization

## 🌳 Dendrogram Insight

The **Dendrogram** helps determine the optimal number of clusters by visualizing distances between data points.
This ensures better segmentation accuracy.

---

## 📌 Customer Segments

The model typically identifies:

* Cluster 1 → Medium Income, Medium Spending
* Cluster 2 → High Income, High Spending
* Cluster 3 → Low Income, Low Spending
* Cluster 4 → High Income, Low Spending
* Cluster 5 → Low Income, High Spending
